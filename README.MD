# POP3 Client

## Описание проекта

Этот проект представляет собой программу для работы с почтовым сервером через протокол POP3. Так как POP3 не предполагает создание и отправку писем на почтовый сервер, для учебных целей был создан локальный почтовый сервер на основе **hMailServer**. А для отправки тестового сообщения на этот сервер был написан скрипт на **Python**. Скрипт использует **SMTP** для отправки письма с простым вложением (файл `.txt`).
